Profile页面:
    1. Get user's all info:
        发送: 
            function: get
            url: /myaccount.html?id=<userId>
            Content-Type: application/json
            value: {
                        无
                    }
        返回:
            {'success': True,
            'email': string,
            'firstname': string,
            'lastname': string,
            'password': string, 
            'address': string, 
            'zipcode': int, 
            'city': string, 
            'state': string, 
            'bankAccount': int,
            'rating': int,
            'isBlocked': int,  // mySQL里没有boolean, 0 - False, 1 - True 
            'isDeleted': int,  // mySQL里没有boolean, 0 - False, 1 - True 
            'isAdmin': int     // mySQL里没有boolean, 0 - False, 1 - True 
            }


    2. Delete account:
        发送: 
            function: post
            url: /deleteaccount
            Content-Type: application/json
            value: {
                        'userId': int
                    }
        返回:
            {'success': True, 'userId': int}

Sign up 页面:

    1. Add account:
        发送: 
            function: post
            url: /addaccount
            Content-Type: application/json
            value: {
                      "email": string,    
                      "firstname": string,
                      "lastname": string,
                      "password": string,
                      "address": string,
                      "zipcode": int,
                      "city": string,
                      "state": string,
                      "bankAccountNumber": string,
                      "isBlocked": int,  -- 0 or 1
                      "isDeleted": int,  -- 0 or 1
                      "isAdmin": int, -- 0 or 1
                    }
        返回:
            {'success': True, 'userId': int}



Sign in 页面：
    
    1. 对比email+密码是否正确:

                    }
        返回:

购物车页面：
    
    1. 获得购物车内所有item的信息：
        发送: 
            function: get
            url: /cart.html?id=<userId>
            Content-Type: application/json
            value: {
                        无
                    }
        返回:
            'success': True
            'items' : 
                [
                    {
                     'success': True,
                     'itemId':int, 
                     'quantity':int, 
                     'isBid': int,        -- 0 or 1
                     'sellerID':sellerID, 
                     'name':name, 
                     'category':category, 
                     'cur_price':cur_price, 
                     'price_step':price_step,
                     'start_time':start_time, 
                     'end_time':end_time, 
                     'flag':flag, 
                     'buy_now':buy_now, 
                     'buy_now_price':buy_now_price,
                     'shipping_cost':shipping_cost, 
                     'description':description, 
                     'url':url, 
                     'status':status, 
                     'photo':photo
                     },
                    {

                    }, ...
                ]


    2. 添加item -- from buy now：  //auction microservice处理add to cart from auction
        发送: 
            function: post
            url: /additemtocartfrombuynow
            Content-Type: application/json
            value: {
                      "userId": int,
                      "itemId": int,
                      "addQuantity": int

                    }
        返回:
            {'success': True, 'userId': int} 


    3. 减少item：(如果isBid = 0 才可以)

        发送: 
            function: post
            url: /deleteitemfromcart
            Content-Type: application/json
            value: {
                      "userId": int,
                      "itemId": int,
                      "deleteQuantity": int
                    }
        返回:
            {'success': True, 'userId': int} 


    4. 点击购买按钮（即买完了购物车内的东西）

        发送: 
            function: post
            url: /checkoutcart
            Content-Type: application/json
            value: {
                      "userId": int
                    }
        返回:
            {'success': True}

            value: {
                        'userId': int
                    }
        返回:
            {'success': True, 'userId': int}


    3. check user is deleted
        发送: 
            function: get
            url: /checkdeleteaccount

                    }
        返回:
            {'success': True, 'value': int}      -- value == 0: not blocked, value == 1: is blocked
